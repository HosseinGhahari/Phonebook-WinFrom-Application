# Phonebook-WinFrom-Application
a Simple PhoneBook Application That Is Based On Windows Form and Ado.Net 


- Please Make Sure You Have Installed Sql Server On Your Machine 🖥
- Then Import Database File From Database Folder Into Sql Server ♻️

I am learning c# now and people always say learn by doing that's why i
started practising c# with doing a actual simple project , this phonebook 
application made me to learn alot of good things about programing

✅ Features of this application :
1- Add , Remove , Edit Contacts 
2- You can Add User To this Application and manage them
3- Every user Has Their own Private Contacts 
4- You can save your contacts inside a txt file 
5- Search ability by name and last name 
6- Application show us live number of contacts 
7- and etc ...
